**编译：计算机将高级语言自动翻译为汇编语言或者是机器语言。**

+ 高级语言：源语言
+ 目标语言：汇编语言或者机器语言



```mermaid
graph TD
    源语言 ==> 预处理器 --经过预处理的源程序 --> 编译器 -- 汇编语言程序 ==> 汇编器 -- 可重定位的机器代码 ==> 链接器-linker/加载器-loader --> 目标机器代码
```

+ 起始位置 + 相对位置 = 绝对位置
+ 加载器：修改可重定向位置，将修改后的指令和数据放到内存中适当的位置。
